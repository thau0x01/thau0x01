## OSCP - Offensive Security Certified Professional

**Issuing Organization**: Offensive Security  
**Date Obtained**: March 2021  
**Credential ID**: OS-XXXXX

### About

The OSCP is a hands-on penetration testing certification that emphasizes practical skills. The certification requires passing a challenging 24-hour practical exam involving real-world penetration testing scenarios.

### Skills Demonstrated

- Network penetration testing
- Web application exploitation
- Buffer overflow exploitation
- Privilege escalation techniques
- Report writing and documentation
